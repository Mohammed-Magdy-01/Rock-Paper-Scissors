# Frontend Mentor - Rock, Paper, Scissors solution

This is my solution to the **Rock, Paper, Scissors** challenge on Frontend Mentor.

The project includes both the classic game mode and the bonus mode (Rock, Paper, Scissors, Lizard, Spock). I also added score statistics, Local Storage support, and a smoother game experience.

## Overview

### The challenge

Users should be able to:

- Play Rock, Paper, Scissors against the computer
- Play the bonus mode (Rock, Paper, Scissors, Lizard, Spock)
- Keep the score after refreshing the browser
- View the optimal layout on different screen sizes

## Features

- 🎮 Classic Rock, Paper, Scissors mode
- 🎮 Bonus mode (Rock, Paper, Scissors, Lizard, Spock)
- 💾 Score saved using Local Storage
- 📊 Game statistics
  - Games Played
  - Wins
  - Draws
  - Losses

- 🎲 Animated computer selection
- 📱 Fully responsive design
- 📜 Rules popup
- 🔄 Play Again functionality
